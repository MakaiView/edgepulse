# Pi Status LED

A reusable status LED service for Raspberry Pi systems.

Designed for homelab and rack-mounted Raspberry Pi deployments.

## Features

- GPIO-controlled status LED
- Boot animation
- Running indicator
- Shutdown animation
- Configuration file
- Systemd service
- Designed for Raspberry Pi OS Bookworm+

## Planned Features

- RGB LED support
- Service health monitoring
- Docker integration
- AdGuard Home monitoring
- Tailscale monitoring
- Temperature alerts
- Network connectivity checks
- Custom LED patterns
- Multiple LED profiles

## Hardware

Currently tested with:

- Raspberry Pi 5
- GPIO18
- Single-color illuminated pushbutton
- Waveshare PoE M.2 HAT+
- UCTRONICS Rack Mount
